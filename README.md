### Prerequisites
- php
- make
- GNU find (recursively find *.md files)

Usage:  `php md2html.php <inputfile> <outputfile>`
Or:     `php rec_md2html.php <inputdir> <outputdir>`

Build tests:    `make all` or `make re`
Remove tests:   `make clean`
